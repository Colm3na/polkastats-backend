# Polkastats backend

> Polkadot network statistics backend

## Build backend (as root)

``` bash
# Change dir
$ cd /usr/local

# Clone this repo
$ git clone https://github.com/Colm3na/polkastats-backend.git

# Change dir
$ cd polkastats-backend

# Install dependencies
$ npm run install

# Serve node
$ node index.js
```
